# NFL Penalty Analytics Project

## Overview

This project investigates whether certain NFL referees systematically call more or fewer penalties after accounting for game context. The goal is to better understand the role of officiating in shaping game outcomes and league fairness.

## Research Question

Do individual referees exhibit consistent differences in penalty rates after controlling for factors such as teams, game situation, and pace of play?

## Data Sources
Play-by-play data from the nflverse (nflfastR), including penalties and game context
Game-level referee assignments from nflreadr::load_officials()

## Motivation

Officiating can influence both competitive balance and fan experience. Quantifying referee tendencies provides insights for league evaluation, transparency, and strategic analysis.
